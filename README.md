# Need for Speed High Stakes Pocket PC (.z3d, .trk) importer
[![Blender](https://img.shields.io/badge/Blender-v3.6_LTS-blue?logo=blender&logoColor=white)](https://www.blender.org/download/lts/3-6/#versions "Download Blender")

A Blender Add-on that enables importing vehicles and tracks from Need for Speed High Stakes Pocket PC (.z3d, .trk) files.

## Requirements
- [Blender 3.6 LTS](https://www.blender.org/download/lts/3-6/)
- [Wahooney's Collection Custom Properties Panel](https://www.dropbox.com/s/16obllozlbueqhk/wahooney_custom_properties_collection.py)

## Installation
Go to `Edit > Preferences > Addons > Install` and either select .zip file or the unzipped `import_nfshs_ppc_models.py` file.

## Location
`File > Import > Need for Speed High Stakes Pocket PC (.z3d, .trk)`
